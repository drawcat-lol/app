# drawcat.lol

this site lets anyone draw a cat on a 256x256 canvas and share it with the world.

## tech stack

-   nextjs
-   shadcn
-   supabase

## getting started

just clone the repo, fill in the values in a `.env.local` file (refer to [`.env.example`](/.env.example)) and run `npm run dev`.
